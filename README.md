# 🏛️ Boralukatiya Buddhist Center

Official website for **Boralukatiya Buddhist Center**, a sacred Buddhist temple and community center located in Kamburupitiya, Sri Lanka.

## 🌟 Organizations Represented

- 🏯 **Boralukatiya Buddhist Center** (බොරලුකැටිය බෞද්ධ මධ්‍යස්ථානය) — Main temple
- 📚 **Siri Dhammasiddhi Dhamma School** (සිරි ධම්මසිද්ධි ධර්ම පාසල) — Buddhist education
- 👥 **Dahami Tharuna Bauddha Kamituwa** (දැහැමි තරුණ බෞද්ධ කමිටුව) — Youth association

## 🚀 Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v3
- **Animations:** Framer Motion, GSAP
- **3D:** Three.js, React Three Fiber
- **UI Components:** Shadcn UI
- **State:** Zustand
- **Forms:** React Hook Form + Zod
- **Backend:** Firebase
- **Media:** Cloudinary
- **Deployment:** Vercel

## 🎨 Design System

- **Primary Color:** Gold `#D4AF37`
- **Secondary Color:** Warm White `#F8F6F1`
- **Accent Color:** Temple Brown `#8B5E3C`
- **Fonts:** Cormorant Garamond, Inter, Noto Sans Sinhala

## 🛠️ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/boralukatiya-buddhist-center.git

# Navigate to project
cd boralukatiya-buddhist-center

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env.local

# Start development server
npm run dev
